# Professional Portfolio Website

A modern, responsive portfolio website built with HTML5, CSS3, and vanilla JavaScript.

LINK: https://haciyev12.github.io/site1101/

## Features

- **Responsive Design**: Mobile-first approach that works seamlessly on all devices
- **Modern Navigation**: Sticky navigation bar with mobile hamburger menu
- **Multiple Pages**: Home, About, Projects, and Resume pages
- **Social Links**: Footer with links to GitHub, Codecademy, LinkedIn, and Twitter
- **Clean Code**: Well-organized, modular code structure

## Pages

### Home Page (`index.html`)
- Profile photo placeholder
- Brief introduction
- Hero section with call-to-action buttons
- Features section highlighting skills

### About Page (`pages/about.html`)
- Background information
- Qualifications section
- Activities and interests

### Projects Page (`pages/projects.html`)
- Project showcase with images
- Project descriptions
- Technology tags
- Includes Project 1 as required
